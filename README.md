# Retail Discount Engine

A JavaScript program that simulates a retail checkout system by applying discounts, calculating cart totals, and updating product inventory.

## Features

* Stores product data using arrays and objects
* Applies category and customer discounts
* Calculates total purchase cost
* Updates inventory after each purchase
* Displays product information in the console

## Technologies Used

* JavaScript
* Arrays and Objects
* Loops
* `switch` Statements
* Conditional Statements

## What I Learned

This project helped me practice using arrays, objects, loops, conditionals, and functions to build a simple retail checkout and inventory system. 
This project also taught me the importance of detail as my first submission had two dashes in front of a line of code, making the code unable to run.
