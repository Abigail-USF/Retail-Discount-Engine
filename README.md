# Retail Discount Engine

A JavaScript program that simulates a retail checkout system by applying discounts, calculating cart totals, and updating product inventory.

## Features

* Stores product data using arrays and objects
* Applies category and customer discounts
* Calculates total purchase cost
* Updates inventory after each purchase
* Displays product information in the console

## Technologies Used

* JavaScript
* Arrays and Objects
* Loops
* `switch` Statements
* Conditional Statements

## How to Run the Project
1. Clone or download the repository
2. Open the project folder in the VS Code.
3. Open 'index.html' in a web browser
4. Open the browser developer tools and select the Console tab to view the results.
